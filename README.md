# Front-Vue
## Requirements
- GraphQL server running with `.env` `MOCKS=TRUE` OR DB docker running and set with `DATABASE_URL=[YOUR_DB_URL]`

## Setup
- `npm i`
- `npm run dev`

## Update source for sentry
`npx @sentry/wizard@latest -i sourcemaps`
